Agent Master Companion

A lightweight macOS menubar utility for browsing and managing agent instruction files across all your projects.

Download DMG View on GitHub
macOS 13+ · Universal Binary (Apple Silicon + Intel)

Agent File Explorer

Scan any project folder and discover agent files — CLAUDE.md, .cursorrules, AGENTS.md, .windsurfrules, and 12+ tools — grouped by tool in a tree view.

User-Level Config

See all your global agent configurations at a glance. Claude Code memory files grouped by project with full read/edit access.

Scratch Pad

Jot down ideas while waiting for agent responses. Notes auto-save and persist across sessions.

Prompt Composer

Select multiple notes, combine them, and copy to clipboard as a single prompt. Useful for assembling context from scattered thoughts.

Privacy First

  • Fully offline — zero network requests
  • No telemetry, no analytics, no data collection
  • Only reads instruction and context files — never touches credentials or config files
  • File content is never cached; only paths and metadata are stored
  • Open source — audit every line of code

Installation

  1. Download the DMG from GitHub Releases
  2. Open the DMG and drag the app to Applications
  3. Double-click to launch — macOS will show a security warning
  4. Open System Settings → Privacy & Security, scroll down, and click "Open Anyway"
  5. The app appears in your menubar (not in the Dock)

The security warning appears because the app is not yet signed with an Apple Developer certificate. This is planned for v1.0. The app is fully open source and can be built from source.
